# Лабораторна робота №4

## Опис проєкту
Цей репозиторій містить контейнеризовану за допомогою Docker математичну модель медичної діагностики за множиною біомаркерів  (5 семестр)

Модель написана мовою Python і використовує бібліотеки `numpy`, `scipy` та `matplotlib` для розрахунків і візуалізації (пошук коренів системи нелінійних рівнянь та калібрування параметрів)
